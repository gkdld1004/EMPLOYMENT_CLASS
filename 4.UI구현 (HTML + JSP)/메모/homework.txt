- html
    : 웹 페이지 전체적인 레이아웃을 잡는 것
    <html> : <head>와 <body>를 모두 묶어주는 역할

- head
    : 내가 보고 있는 웹페이지의 정보를 모아둔 태그, 문서 정보 정의
    <meta> : 웹 페이지의 제목과 내용, 중요한 keyword가 들어있는 정보태그
    <title> : 웹 페이지 상단바의 제목

- body
    : HTML 안의 몸이라고 생각. 텍스트, 이미지, 링크, 목록, 테이블 등 문서의 주요 내용이 보일 영역 지정

- form
    : 사용자가 입력하거나 선택한 정보를 서버로 전송하기 위해 사용

- input
    : 사용자가 데이터를 입력할 수 있는 입력 필드를 지정하는 요소
(form이 서버에 데이터를 전송해주는 역할이라면, input은 그 데이터를 입력할 수 있는 공간을 제공해주는 역할)

	Type
	   "text" : text 창을 의미
	   "password" : 암호 창을 의미
	   "submit" : 데이터를 제출하는 창을 의미, 클릭시 서버로 데이터가 넘어감 (Button과 비슷)
	   "radio" : 선택 항목을 표시하여, 사용자가 그 항목 중 하나를 선택할 수 있게함
	   "checkbox" : 체크박스를 만들어 사용자가 여러 항목을 체크할 수 있도록 함
- label
    : 입력 창의 설명

- select
    : 콤보박스. option태그와 같이 사용.

- textarea
    : 여러 줄의 텍스트를 입력할 수 있는 폼 요소
    rows - 아래줄의 수 / cols - 옆의 칸 수 

- ul
    : unordered list, 순서가 없는 목록태그형태. 
    각 리스트들 앞에 순번이 아닌 불릿(●, ■등)

- ol
    : ordered list, 순서가 있는 목록태그형태.
    각 리스트들 앞에 순번이 주어진다

- li
    : list, 독립적 사용이 불가능. 따라서, <ul> or <ol>과 함께 사용

- fieldset
    : <form>요소에서 연관된 요소들을 하나의 그룹으로 묶을 때 사용
    하나의 그룹으로 묶은 요소들 주변으로 박스 모양의 선을 그려줌

- legend
    : <fieldset> 요소의 캡션(caption)을 정의할 수 있음
    <fieldset>태그로 묶은 요소들에 제목을 달 때 사용

- img
    : image, 웹페이지에 이미지를 삽입
    src : source, 이미지의 원본 소스의 위치 속성을 넣어줌
    alt : 이미지를 표시할 수 없을 경우 대신 텍스트 설명으로 대체하기 위한 속성
    width : 이미지의 가로 사이즈 기재(숫자만 입력 시 픽셀값으로 인지)
    heigt : 이미지의 세로 사이즈 기재(숫자만 입력 시 픽셀값으로 인지)

- video
    : 웹페이지에서 영상파일을 재생할 때 사용
    src : 경로
    controls : 컨트롤러 표시 (플레이/정지 등의 버튼이 있는 제어기 표시)
    autoplay : 동영상 자동 재생
    loop : 동영상 반복 재생
    width : 영상의 너비(가로길이)
    heigt : 영상의 높이(세로길이)
    muted : 동영상 음소거

- table
    : 행과 열로 이루어진 표

- tr
    : table row, 테이블에서 행을 구분

- td
    : table data, 테이블의 열을 각각의 셀(cell)로 나누어 줌

- th
    : table header, 각 열의 제목을 나타내며, 모든 내용은 자동으로 굵은 글씨에 가운데 정렬

- a
    : anchor, 링크를 생성하여 원하는 홈페이지에 링크를 걸어 이동할 수 있도록 해줌
    href="url", target="_self(지금 보고 있는 화면으로) / _blank(새로운 창) / _parent(새로운 탭)"

- button
    : 클릭 가능한 버튼을 나타냄.
    button - 클릭할 수 있는 기능
    submit - 데이터를 전송할 수 있는 기능
    reset - 데이터를 초기화할 수 있는 기능

- h1 ~ h6
    : HTML 제목을 정의하는데 사용
    <h1> - 가장 중요한 제목 정의(페이지 당 하나만 사용) , <h6> 가장 덜 중요한 제목을 정의

- p
    : 문단 구분 태그, 줄바꿈 포함

- br
    : 줄바꿈 태그

- hr
    : A horizontal rule, 웹페이지에 가로로 선을 긋는 태그. 마무리태그X

- blockquote
    : 어떤 부문을 전체 그대로 인용할 때 쓰는 태그
    한줄을 넘어가는 긴 문장이나 여러문장쓸때

- div
    : division, 여러 요소들을 하나로 묶어서 구분
    요소들을 묶어놓으면 배경색이나 테두리 등의 스타일을 구역별로 적용하기 쉬움, 웹페이지 레이아웃을 만들 때 가장 많이 쓰는 태그

- span
    : 문서의 구조를 나타내는 태그가 아니라, 단지 일부 내용/요소를 구분하기 위한 목적의 태그
    주로 특정 부분에 스타일을 적용시킬 때 사용
