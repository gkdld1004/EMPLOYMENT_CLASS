Input 요소

input 태그는 사용자가 데이터를 입력할 수 있도록 제공하는 UI 요소이다

<input type="??"> ?? 에 따라서 화면에 표시되거나, 사용자가 선택하는 방식이 달라진다

<input type="text">

input에 type을 지정하지 않으면 text로 작동한다
문자열 정보를 사용자에게 입력받을 수 있도록 텍스트 필드를 만들어낸다






<input type="password"> type="text" 와 같으나, 입력하는 글자를 노출시키지 않는다
단, 서버에 전송하는 경우 데이터가 드러날 수 있다 (암호화는 아니다)
<input type="checkbox"> 여러 항목에 대해, 사용자가 다중선택이 가능한 보기를 나열한다
